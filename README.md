## 使用
先安装 `pnpm`

```sh
npm i -g pnpm
```

安装依赖
```sh
pnpm install
```

开发启动
```sh
pnpm dev
```

构建
```sh
pnpm build
```

预览产物
```sh
pnpm preview
```
