# 第 31 本《就是想看你笑的样子》

## 简介
“如果生活不能让你笑，你就笑笑生活。”这是一本纯手绘漫画书，故事来源于懒兔子与身边亲友的真实生活，充满了浓浓的爱与真情，温馨治愈。在诙谐、幽默的对话中，治愈我们心中的烦闷、压抑，安慰身处焦虑中的你我。每一个看过这本书的人，都会露出最真挚的笑容。
