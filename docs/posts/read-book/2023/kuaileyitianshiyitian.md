# 第 33 本《快乐一天是一天》

## 简介
超人气原创动漫博主“小蓝和他的朋友”重磅新作，爆笑生活冷知识漫画合辑，在哈哈大笑中收获健康快乐生活小秘诀！

熬夜的时候，“僵尸”竟然吃掉了小蓝的脑子？！聪慧机智的小紫，学习时学着学着突然就“不识字”了，甚至怀疑自己痴呆了！热爱美食的G胖为了神不知鬼不觉地大吃特吃，花光所有私房钱购入了一件超级隐身衣……

围观小蓝、G胖、小红、小紫的爆笑生活日常，和宠物猫小花一起快乐“吃瓜”，“解锁”平凡生活中的趣味冷知识
